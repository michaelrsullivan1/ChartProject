# User Podcast Pages Scope (High-Level)

Date: 2026-04-22

## Purpose

This note locks the high-level product intent for the three new user podcast pages.

It is intentionally non-technical and should be used as a scope reference before
implementation details are added.

Current pilot user:

- Michael Saylor (`michael-saylor`)

## Page 1: User Narrative Mix

Goal:

- show how the mix of Michael Saylor's narratives changes over time

Primary question:

- what was he mostly talking about in each period

Expected value:

- makes narrative shifts visible by composition, not only by raw counts

## Page 2: User Narrative Intensity

Goal:

- show how strongly and how often Michael Saylor's belief output appears over time

Primary question:

- when was his podcast belief activity most active or concentrated

Expected value:

- separates narrative composition changes from simple activity spikes

## Page 3: User Beliefs

Goal:

- show representative belief and quote examples that explain the trend views

Primary question:

- what was he actually saying during each period and narrative phase

Expected value:

- provides direct textual evidence so trend views stay interpretable

## How The Three Pages Work Together

1. User Narrative Mix: what changed
2. User Narrative Intensity: how strongly or frequently it changed
3. User Beliefs: what the change looked like in actual language

## Scope Guardrails

- keep this podcast work separate from tweet/X feature logic
- keep person-first focus for the pilot phase
- avoid UI polish decisions until usefulness is validated
- treat this note as the high-level scope lock for this phase
